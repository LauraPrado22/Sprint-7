# Sprint-7
# Análisis de Vehículos en Venta en EE.UU. 

## Descripción
Esta aplicación web interactiva fue desarrollada con **Streamlit** y permite explorar 
un conjunto de datos de anuncios de venta de coches en Estados Unidos.

## Funcionalidad
- 📊 **Histograma**: Muestra la distribución del odómetro de los vehículos en venta.
- 🔵 **Gráfico de dispersión**: Visualiza la relación entre el precio y el odómetro.
- El usuario puede seleccionar qué gráfico visualizar mediante casillas de verificación.

## Tecnologías utilizadas
- Python
- Streamlit
- Plotly
- Pandas

## Cómo ejecutar la aplicación
```bash
streamlit run app.py
```
